## 2. R 기본 문법

### 2.1 변수와 데이터 타입

```r
x <- 10        # 할당 연산자 `<-` 사용
y = 20         # `=`도 가능
```

```r
v <- c(1, 2, 3, 4)
```

| 구조       | 설명                            | 예제                                  |

## 6. 데이터 처리 (dplyr 패키지)

```r
library(dplyr)

df %>%
  filter(age > 30) %>%
  select(name, age) %>%
  arrange(desc(age))
```

- `filter`: 행 조건
- `select`: 열 선택
- `arrange`: 정렬
- `mutate`: 새로운 변수 생성
- `summarise`: 요약 통계

------

## 7. 시각화 (ggplot2)

```r
library(ggplot2)

ggplot(df, aes(x=age, y=salary)) +
  geom_point() +
  geom_smooth(method="lm")
```

- `geom_point`: 산점도
- `geom_bar`: 막대그래프
- `geom_line`: 선그래프

------

## 8. 통계 분석 기초

```r
# 평균, 분산
mean(df$age)
var(df$age)

# 상관관계
cor(df$age, df$salary)

# 회귀분석
model <- lm(salary ~ age, data=df)
summary(model)
```

# 📑 R 기초 핵심 요약집 (압축본)

## 1. 기본 문법

- **할당**: `x <- 10` (또는 `x = 10`)
- **데이터 타입**: `numeric`, `integer`, `character`, `logical`, `factor`
- **형 변환**: `as.numeric()`, `as.character()`, `as.factor()`
- **출력**: `print()`, `cat()`

------

## 2. 자료구조

- **Vector**: `c(1,2,3)`
- **Matrix**: `matrix(1:6, nrow=2)`
- **List**: `list(1,"a",TRUE)`
- **Data Frame**: `data.frame(x=1:3, y=c("a","b","c"))`
- **Factor**: 범주형 데이터 → `factor(c("M","F","M"))`

------

## 3. 제어문 & 함수

- **조건문**: `if / else / else if`

- **반복문**: `for`, `while`, `repeat`

- **함수 정의**:

  ```r
  f <- function(x) { return(x^2) }
  ```

- **apply 계열**:

  - `apply()`: 행/열 단위 연산
  - `lapply()`: 리스트에 함수 적용
  - `sapply()`: 벡터/행렬 반환

------

## 4. 데이터 입출력

- **CSV 읽기**: `read.csv("file.csv")`
- **엑셀 읽기**: `library(readxl); read_excel("file.xlsx")`
- **데이터 확인**: `head()`, `str()`, `summary()`

------

## 5. 데이터 처리 (dplyr)

- `filter()`: 행 조건
- `select()`: 열 선택
- `arrange()`: 정렬
- `mutate()`: 새 변수 생성
- `summarise()`: 요약 통계
- `%>%`: 파이프 연산자

------

## 6. 시각화 (ggplot2)

- 기본 구조:

  ```r
  ggplot(df, aes(x, y)) + geom_point()
  ```

- 주요 함수:

  - `geom_point()`: 산점도
  - `geom_bar()`: 막대그래프
  - `geom_line()`: 선그래프
  - `geom_histogram()`: 히스토그램

------

## 7. 통계 분석 기초

- **기술통계**: `mean()`, `median()`, `var()`, `sd()`
- **상관관계**: `cor(x,y)`
- **회귀분석**: `lm(y ~ x, data=df)` → `summary()`

- **가설검정**: `t.test(x,y)`, `chisq.test()`

# 📑 10. 시험 포인트 (상세 설명)

### 1) 기본 문법 (`<-`, 자료구조 차이)

- **출제 이유**: R은 다른 언어와 달리 `<-`를 기본 할당 연산자로 사용합니다.
- **시험 포인트**: “R에서 변수 할당 연산자는?” 같은 단답형 문제로 자주 등장.
- **자료구조 차이**: Python의 리스트와 R의 벡터, 데이터프레임 개념 비교 문제 출제 가능.

------

### 2) `apply` 계열 함수

- **출제 이유**: R의 반복 처리 핵심.
- **시험 포인트**: `apply`, `lapply`, `sapply`의 차이 구분.
  - `apply`: 행/열 단위 연산
  - `lapply`: 리스트 반환
  - `sapply`: 벡터/행렬 반환
- **예상 문제**: “리스트에 함수를 적용하고 결과를 벡터로 반환하는 함수는?”

------

### 3) `dplyr` 주요 함수

- **출제 이유**: 데이터 처리 패키지 중 가장 많이 쓰임.
- **시험 포인트**: `filter`, `select`, `arrange`, `mutate`, `summarise` 기능 구분.
- **예상 문제**: “데이터프레임에서 조건에 맞는 행만 추출하는 함수는?”

------

### 4) `ggplot2` 기본 문법

- **출제 이유**: 시각화는 ADP 시험의 필수 영역.
- **시험 포인트**: `ggplot(df, aes(...)) + geom_*` 구조 이해.
- **예상 문제**: “산점도를 그릴 때 사용하는 geom 함수는?”

------

### 5) 기술통계·회귀·분산분석 함수
