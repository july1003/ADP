# 📘 ADP 통계 검정 선택 맵 (문제 → 검정법)

------

## 1️⃣ 평균 비교 문제 (가장 자주 나옴)

### 🔹 ① 한 집단 평균 vs 기준값

**문제 유형**

- “A 공정의 평균 불량률이 기준값과 다른지 검정하시오”

| 조건                       | 검정                          |
| -------------------------- | ----------------------------- |
| 정규성 만족, 모분산 알려짐 | **z-test**                    |
| 정규성 만족, 모분산 미지   | **one-sample t-test**         |
| 정규성 불만족              | **Wilcoxon signed-rank test** |

**서술형 핵심 문장**

> 표본 수가 적고 모분산을 알 수 없으므로 t-test를 사용한다.

------

### 🔹 ② 두 집단 평균 비교 (ADP 최빈)

**문제 유형**

- “남녀 간 평균 소득 차이가 있는지 검정하시오”

| 조건              | 검정                              |
| ----------------- | --------------------------------- |
| 독립, 정규성 만족 | **independent two-sample t-test** |
| 분산 다름         | **Welch’s t-test**                |
| 정규성 불만족     | **Mann–Whitney U test**           |
| 대응 표본         | **paired t-test**                 |

**출제 포인트**

- 독립 vs 대응 구분 ⭐️⭐️⭐️

------

## 2️⃣ 세 집단 이상 평균 비교

**문제 유형**

- “세 지역 간 평균 만족도 차이를 검정하시오”

| 조건               | 검정                    |
| ------------------ | ----------------------- |
| 정규성·등분산 만족 | **ANOVA**               |
| 정규성 불만족      | **Kruskal–Wallis test** |

📌 **추가 질문**

> “ANOVA 유의 → 어느 집단이 다른가?”
> → **사후검정(Post-hoc test)** 필요 (Tukey 등)

------

## 3️⃣ 비율·빈도 비교 (범주형 데이터)

### 🔹 ③ 범주형 변수 간 관계

**문제 유형**

- “흡연 여부와 질병 발생 간 관련성 검정”

| 조건         | 검정                          |
| ------------ | ----------------------------- |
| 표본 수 충분 | **Chi-square test (χ² test)** |
| 기대빈도 < 5 | **Fisher’s exact test**       |

**서술형 문장**

> 범주형 변수 간 독립성을 검정하기 위해 카이제곱 검정을 사용한다.

------

## 4️⃣ 분산 비교 (ADP에서 가끔 등장)

**문제 유형**

- “두 공정의 분산이 동일한지 검정하시오”

| 조건          | 검정              |
| ------------- | ----------------- |
| 정규성 만족   | **F-test**        |
| 정규성 불만족 | **Levene’s test** |

------

## 5️⃣ 상관관계 분석

**문제 유형**

- “두 변수 간 상관관계가 존재하는지 분석하시오”

| 조건           | 검정                          |
| -------------- | ----------------------------- |
| 정규성, 연속형 | **Pearson correlation**       |
| 비정규 / 순위  | **Spearman rank correlation** |

📌 **주의**

> 상관관계 ≠ 인과관계

------

## 6️⃣ 회귀 분석 관련 검정 (ADP 심화)

### 🔹 회귀계수 유의성

- **t-test**: 개별 계수
- **F-test**: 전체 모델

### 🔹 다중공선성 진단

- **VIF (Variance Inflation Factor)**
  - VIF > 10 → 문제 있음

------

## 7️⃣ 정규성 검정 (선택 근거 제시용)

**문제 유형**

- “정규성 가정을 확인하시오”

| 방법                    |
| ----------------------- |
| Shapiro–Wilk test       |
| Kolmogorov–Smirnov test |
| Q-Q plot                |

📌 **서술형**

> 정규성 가정이 만족되지 않아 비모수 검정을 적용하였다.

------

## 8️⃣ 시험장에서 쓰는 **결정 트리 사고 흐름** ⭐️⭐️⭐️

아래 순서로 생각하면 **절대 틀리지 않습니다**.

```
① 평균인가? 비율인가?
② 집단 수는 몇 개인가?
③ 정규성 만족하는가?
④ 독립인가? 대응인가?
```

👉 이 4단계만 판단하면 검정 선택 끝

------

- “두 집단이 독립적이므로 독립표본 t-test를 적용한다.”
- “범주형 변수 간 독립성을 검정하기 위해 카이제곱 검정을 사용한다.”
- “ANOVA 이후 사후검정을 통해 집단 간 차이를 확인한다.”

# 📝 ADP 통계 검정 선택 실전 문제 15제

------

## 1️⃣ 문제 1 — 단일 집단 평균

**문제**
한 공정의 평균 불량률이 기준값 5%와 다른지 검정하고자 한다.
표본 수는 30이며 모분산은 알려져 있지 않다.

### ✅ 정답

- **검정**: one-sample t-test

### ✍️ 서술형 모범답안

> 모분산을 알 수 없고 표본 수가 제한적이므로 단일 표본 t-test를 사용한다.

------

## 2️⃣ 문제 2 — 단일 집단, 비정규

**문제**
소규모 표본에서 측정된 작업 시간이 정규성을 만족하지 않는다.
평균이 기준값과 다른지 검정하시오.

### ✅ 정답

- **검정**: Wilcoxon signed-rank test

### ✍️ 서술형

> 정규성 가정을 만족하지 못하므로 비모수 검정을 적용한다.

------

## 3️⃣ 문제 3 — 두 집단 평균, 독립

**문제**
남성과 여성의 평균 연봉 차이가 있는지 분석하고자 한다.
두 집단은 서로 독립이며 정규성을 만족한다.

### ✅ 정답

- **검정**: independent two-sample t-test

------

## 4️⃣ 문제 4 — 두 집단 평균, 분산 다름

**문제**
두 지역의 평균 만족도를 비교한 결과 분산이 서로 다름이 확인되었다.

### ✅ 정답

- **검정**: Welch’s t-test

### ✍️ 서술형

> 두 집단의 분산이 동일하지 않으므로 Welch의 t-test를 적용한다.

------

## 5️⃣ 문제 5 — 두 집단, 비정규

**문제**
두 브랜드의 고객 대기시간 분포가 비정규성을 보인다.

### ✅ 정답

- **검정**: Mann–Whitney U test

------

## 6️⃣ 문제 6 — 대응 표본

**문제**
다이어트 전·후 체중 변화가 유의미한지 분석하시오.

### ✅ 정답

- **검정**: paired t-test

### ✍️ 서술형

> 동일 대상의 전후 비교이므로 대응표본 t-test를 사용한다.

------

## 7️⃣ 문제 7 — 대응 표본, 비정규

**문제**
전·후 만족도 점수가 정규성을 만족하지 않는다.

### ✅ 정답

- **검정**: Wilcoxon signed-rank test

------

## 8️⃣ 문제 8 — 세 집단 이상 평균

**문제**
세 지역 간 평균 매출 차이를 분석하고자 한다.
정규성과 등분산 가정이 충족된다.

### ✅ 정답

- **검정**: ANOVA

------

## 9️⃣ 문제 9 — 세 집단 이상, 비정규

**문제**
세 교육 방식 간 시험 점수 분포가 비정규이다.

### ✅ 정답

- **검정**: Kruskal–Wallis test

------

## 🔟 문제 10 — 사후검정

**문제**
ANOVA 결과 유의미한 차이가 확인되었다.
어느 집단 간 차이인지 확인하려면?

### ✅ 정답

- **검정**: Post-hoc test (Tukey 등)

### ✍️ 서술형

> 분산분석 이후 집단 간 차이를 확인하기 위해 사후검정을 수행한다.

------

## 1️⃣1️⃣ 문제 11 — 범주형 변수 간 관계

**문제**
흡연 여부와 질병 발생 간 관련성이 존재하는지 분석하시오.

### ✅ 정답

- **검정**: Chi-square test (χ² test)

------

## 1️⃣2️⃣ 문제 12 — 기대빈도 작음

**문제**
일부 셀의 기대빈도가 5 미만이다.

### ✅ 정답

- **검정**: Fisher’s exact test

------

## 1️⃣3️⃣ 문제 13 — 분산 비교

**문제**
두 공정의 분산이 동일한지 검정하고자 한다.
정규성을 만족한다.

### ✅ 정답

- **검정**: F-test

------

## 1️⃣4️⃣ 문제 14 — 상관관계

**문제**
두 연속형 변수 간 선형 상관관계를 분석하시오.
정규성을 만족한다.

### ✅ 정답

- **검정**: Pearson correlation

------

## 1️⃣5️⃣ 문제 15 — 상관관계, 비정규

**문제**
순위 데이터 간 상관관계를 분석하시오.

### ✅ 정답

- **검정**: Spearman rank correlation
