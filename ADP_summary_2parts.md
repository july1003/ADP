# 📘 ADP에만 있는 2개 과목 집중 해설

## ✅ ADP 전용 과목 구성

ADP 필기에는 ADsP에 없는 **다음 2개 과목**이 있습니다.

1. **데이터 분석 기획 (심화)**
2. **데이터 분석 활용 (심화)**

이 두 과목은 **기술 지식이 아니라 ‘전문가 관점’을 평가**합니다.

------

## Ⅰ. 데이터 분석 기획 (ADP 심화 과목)

### 🔍 이 과목의 출제 의도

> “이 사람은 분석을 *할 줄 아는가*가 아니라
> **분석을 ‘설계하고 실패를 예방할 수 있는가’**를 본다.”

즉,

- ADsP: *분석이 무엇인지 아는가*
- ADP: *분석 프로젝트를 책임질 수 있는가*

------

### 1️⃣ 분석 기획의 핵심 구성 요소 (ADP 필수)

#### (1) 문제 정의

- 현상(problem) ≠ 분석 문제
- **의사결정 질문으로 재정의**

📌 예

- 현상: “매출이 감소했다”
- 분석 문제: “어떤 요인이 매출 감소에 가장 큰 영향을 미치는가”

------

#### (2) 분석 목표와 KPI

- 분석 목표는 **측정 가능**해야 함
- KPI는 **분석 결과의 성공 기준**

📌 ADP 포인트

> KPI가 불명확하면 분석은 실패한다.

------

#### (3) 데이터 확보 가능성

- 데이터 존재 여부
- 데이터 품질
- 수집 비용과 시간

📌 서술형 문장

> 분석 기획 단계에서는 데이터 확보 가능성과 품질을 사전에 검토해야 한다.

------

### 2️⃣ 분석 방법론을 “왜” 선택하는가

ADsP: 방법론 나열
ADP: **선택 이유 설명**

#### CRISP-DM이 가장 많이 쓰이는 이유

- 비즈니스 중심
- 반복 가능
- 실무 적용 용이

📌 고득점 문장

> CRISP-DM은 비즈니스 이해를 출발점으로 하여 분석 결과의 실무 적용까지 고려한 방법론이다.

------

### 3️⃣ 분석 프로젝트 실패 원인 (ADP 단골)

| 실패 원인      | 설명                |
| -------------- | ------------------- |
| 문제 정의 오류 | 분석 결과 활용 불가 |
| KPI 부적절     | 성과 측정 불가      |
| 데이터 부족    | 모델링 불가         |
| 현업과 괴리    | 결과 미사용         |

📌 단골 문장

> 분석 프로젝트의 실패 원인은 기술적 한계보다 기획 단계의 문제인 경우가 많다.

------

### ✍️ 이 과목 서술형 공식

```
문제 정의 → 목표/KPI → 데이터 → 방법론 → 활용
```

이 흐름이 보이면 **고득점**입니다.

------

## Ⅱ. 데이터 분석 활용 (ADP 심화 과목)

### 🔍 이 과목의 출제 의도

> “모델을 만들었을 때
> **그 결과를 실제 의사결정에 쓸 수 있는가**?”

즉,

- ADsP: 모델 설명
- ADP: **모델 결과의 활용·확산·통제**

------

### 1️⃣ 분석 결과 활용 유형 (ADP 핵심)

#### (1) 의사결정 지원

- 보고서
- 대시보드
- 전략 수립

#### (2) 업무 자동화

- 추천 시스템
- 이상 탐지
- 예측 시스템

📌 서술형 문장

> 분석 결과는 의사결정 지원뿐 아니라 업무 자동화에 활용될 수 있다.

------

### 2️⃣ 모델 성능보다 중요한 것 (ADP 관점)

| 요소        | 이유      |
| ----------- | --------- |
| 해석 가능성 | 현업 수용 |
| 안정성      | 운영 환경 |
| 재현성      | 지속 사용 |

📌 고급 문장

> 실무에서는 모델 성능뿐 아니라 해석 가능성과 운영 안정성이 중요하다.

------

### 3️⃣ 분석 결과 확산과 관리

- 모델 모니터링
- 성능 저하 감지
- 재학습 전략

📌 ADP 포인트
→ **“모델은 한 번 만들고 끝이 아니다”**

------

### 4️⃣ 윤리·보안·책임 (ADP 전용 시각)

- 개인정보 보호
- 데이터 오남용 방지
- 결과 책임성

📌 단골 문장

> 데이터 분석 결과의 활용에는 윤리적 고려와 책임이 수반되어야 한다.

------

## ✍️ 데이터 분석 활용 서술형 공식

```
활용 목적 → 적용 방식 → 기대 효과 → 유의사항
```
