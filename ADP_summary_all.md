- 산출물 정의

- 도입 → 확산 → 정착 → 고도화

## Ⅲ. 데이터 분석 – 통계 (Statistics) ⭐️⭐️⭐️

### 7️⃣ 기술통계

- 평균, 중앙값, 분산, 표준편차
- 왜도, 첨도
- 이상치 영향

------

### 8️⃣ 추론통계 & 가설검정 ⭐️⭐️⭐️

#### 가설검정 절차

1. 귀무가설(H₀)
2. 대립가설(H₁)
3. 유의수준(α)
4. 검정통계량
5. p-value 판단

**p-value 정의 (정확 버전)**

> p-value는 귀무가설이 참이라는 가정 하에서, 관측된 통계량과 같거나 그보다 더 극단적인 값이 발생할 확률이다.

------

### 9️⃣ 모수적 vs 비모수적 검정 (ADP 차별)

- 모수: t-test, ANOVA
- 비모수: Mann-Whitney, Wilcoxon

------

### 🔟 다중공선성

- 독립변수 간 강한 상관관계
- VIF > 10 → 문제
- 해결: 변수 제거, PCA, Ridge

------

## Ⅳ. 데이터 분석 – 머신러닝 / 데이터마이닝 ⭐️⭐️⭐️

### 1️⃣1️⃣ 회귀 / 분류

- 선형 회귀: 연속값 예측
- 로지스틱 회귀: 확률 기반 분류

------

### 1️⃣2️⃣ 주요 분류 알고리즘

- Decision Tree: 직관적, 과적합
- Random Forest: 앙상블, 분산 감소
- SVM: 마진 최대화
- KNN: 거리 기반

**단골 서술**

> Random Forest는 다수의 트리를 결합하여 분산을 감소시킴으로써 과적합을 완화한다.

------

### 1️⃣3️⃣ 앙상블 기법

- Bagging: 분산 감소
- Boosting: 오분류 보완
- Stacking: 메타 모델

------

### 1️⃣4️⃣ 차원 축소

- PCA: 분산 최대화, 해석 어려움
- LDA: 클래스 분리 최대화

------

### 1️⃣5️⃣ 정규화(Regularization)

- Ridge(L2): 계수 축소
- Lasso(L1): 변수 선택

------

## Ⅴ. 모델 평가 & 검증 ⭐️⭐️⭐️

### 1️⃣6️⃣ 분류 평가 지표

- Accuracy (불균형 취약)
- Precision
- Recall
- F1-score
- ROC-AUC

**단골 문장**

> 불균형 데이터에서는 Accuracy가 모델 성능을 왜곡할 수 있다.

------

### 1️⃣7️⃣ ROC Curve vs PR Curve

- ROC: TPR–FPR
- PR: Precision–Recall
- 극단적 불균형 → PR Curve 적합

------

### 1️⃣8️⃣ 교차검증

- K-fold
- Stratified K-fold
- LOOCV
